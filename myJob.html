I used to be a computer science tutor in CDE college (now Cestar College) in Sherbrooke, QC. The courses I have given include:

  • Older operating system – a general introduction to the manilupation of vintage OS (Windows XP), the file system, the command line to manipulate the file system, and batch file programming.

  • Application Software – a general guide to the basic functionalities and manipulations of the Microsoft Office suites, includes the Microsoft Word, Access, Excel, and PowerPoint.

  • Database – It's all about a basic introduction of the Relational Database Management System - RDBMS, the Schemas, ER-Model, Keys, INNER JOIN, OUTER JOIN, etc. And the creation and manipulation of a sample databases in MySQL.

  • Programming Utility Software – It's a guide to the programming of basic utility software by Visual Basic.

  • Computer Network: Access Management & Resource Sharing – This course is an introduction to the computer network in layered architecture, which includes the OSI model, the TCP/IP model, the protocols of each layer.
